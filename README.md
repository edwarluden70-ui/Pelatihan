# Aplikasi Pelatihan Mengetik Tunanetra - BPVP Sorong

Sistem aplikasi berbasis web (*e-learning*) interaktif yang dikembangkan secara khusus untuk mendampingi rekan-rekan tunanetra dalam menguasai keterampilan mengetik sepuluh jari secara taktil (*touch typing*).

## 📌 Atribusi Proyek
* **Institusi:** Balai Pelatihan Vokasi dan Produktivitas (BPVP) Sorong
* **Arsitek & Pengembang Sistem:** Helmi Palpialy S.T

## 🚀 Fitur Unggulan
1. **Kurikulum Berbasis Kompetensi:** Pembagian paket fleksibel mulai dari **40 JP (Dasar)**, **80 JP (Menengah)**, hingga **120 JP (Mahir)**.
2. **Dashboard On-Demand:** Tampilan kurikulum dinamis yang terkunci di awal dan hanya akan memuat unit kompetensi saat paket JP dipilih.
3. **Teknologi Tanpa Server (Serverless TTS):** Menggunakan integrasi *Web Speech API* bawaan peramban (*browser*) untuk mendiktekan instruksi secara lancar dalam Bahasa Indonesia tanpa memerlukan aset audio eksternal yang besar.
4. **Desain Aksesibilitas Tinggi:** Palet warna biru cerah berkontras tinggi serta elemen antarmuka berukuran besar yang ramah terhadap instruktur maupun pengguna *low vision*.
